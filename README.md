#  Ascend

> Where your thoughts become clarity.

Ascend is a modern personal journaling and self-reflection application built with React. It provides a beautiful, distraction-free space for writing daily entries, tracking emotions, reflecting on personal growth, and gaining insights from journaling habits.

---

## Features

###  Journal Writing
- Create and save daily journal entries
- Clean, focused writing experience

### Mood Tracking
- Record your mood with each entry
- Multiple mood categories:
  - Joyful
  - Grateful
  - Reflective
  - Calm
  - Excited
  - Tired
  - Anxious
  - Uncertain

### Entry Management
- Browse all journal entries
- View entry previews
- Organize entries with tags
- Read full journal entries

### Insights Dashboard
- Mood distribution analytics
- Writing activity tracking
- Journaling streak statistics
- Personal growth metrics

### Profile Section
- User profile overview
- Privacy settings
- Export options
- Appearance preferences
